Project in Progress

DarkRP In Unreal

Project Information

Engine: UE5
State: Early Prototype
Goals: Modularity, PCG, Engagement

Core Concept

DarkRP in Unreal is a systems-driven prototype inspired by the Garry’s Mod DarkRP framework, reimagined within Unreal Engine 5. The project focuses on building a modular roleplay foundation featuring player jobs, interactive entities, inventory and economy systems, and administrative tools. My current development phase centres on backend gameplay logic, including entity interaction, analytics tracking, and UI design for dynamic player management.

Key Features (In Progress)

Modular job and inventory systems
Custom analytics tracking (player statistics and progression)
Interactive entities with in-world feedback
Functional admin interface for player and system monitoring
Designed for scalability and multiplayer integration

Focus Area

Technical system design, modularity, and usability within a multiplayer environment.
