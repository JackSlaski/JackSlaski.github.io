UI Showcase

A UI Showcase Project made on an Unreal Engine 5 RPG framework. This project was my first UE5 UI project and it really helped me realise and focus on my love for UI in games. I attempted to make the UI as modular and as technically sound as possible to maximise engine use and efficiency.

Features

  • Main Game UI and HUD
  • Player Stats
  • Player Abilities
  • Shop System
  • World Object UI
  • Spatial Enemy and Player Health
  • Damage and Level Popups
  • Inventory System
  • Minimap
  • Powerups System
  • Attributes and Upgrading
  • Pause Game
  • Options and Settings Menu
  • Currency System and Animation
